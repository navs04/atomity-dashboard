# Cloud Cost Analytics Dashboard

## Feature Chosen
I recreated the interactive cluster analytics dashboard (Option A).

## Tech Stack
- React
- TypeScript
- Framer Motion
- TanStack Query
- CSS Modules

## Features
- Interactive cluster selector
- Dynamic bar chart
- Data fetched from DummyJSON API
- React Query caching
- Responsive layout
- Scroll-triggered animations
- Hover animations
- Design tokens with CSS variables
- Loading and error states

## Architecture
- Reusable components
- Custom hook for API fetching
- Parent-managed state
- CSS Modules

## Trade-offs
- Used DummyJSON to simulate cloud metrics.
- Focused on animation quality and component architecture

## Future Improvements
- Real cloud metrics API
- Filter dropdowns
- Dark mode
- More detailed charts
